# v1.0.0 — 首个稳定版 / First Stable Release

专业级、开源、安全的影视数据备份工具 / Professional-grade, open-source secure backup for film production.

## 亮点 | Highlights

- **原子复制引擎**：零部分写入，断电安全，并支持检查点恢复 / **Atomic copy engine**: zero partial writes, power-loss safe, with checkpoint recovery
- **多哈希校验**：支持 XXH3、XXH64、XXH128、MD5、SHA-256，并带有 `SkipIfVerified` 智能去重 / **Multi-hash verification**: XXH3, XXH64, XXH128, MD5, and SHA-256 with `SkipIfVerified` smart dedup
- **ASC MHL 链路追踪**：生成行业标准的交接清单 / **ASC MHL chain-of-custody**: industry-standard manifest generation
- **级联复制**：先写入主 SSD，再异步同步到接力盘 / **Cascade copy**: primary SSD first, then asynchronous sync to shuttle drives
- **冲突处理**：支持 Skip / Overwrite / Keep Both，并带有完整哈希比对 / **Conflict resolution**: Skip / Overwrite / Keep Both with full hash comparison
- **自动重试与复验**：失败文件自动重试，并重新哈希确认 / **Auto-retry with re-verify**: failed files retry automatically, then re-hash to confirm
- **崩溃恢复**：支持从任意中断点恢复，并使用强制覆盖保证安全 / **Crash recovery**: resume from any interruption point with force-overwrite for safety
- **实时进度**：提供单文件速度、ETA 和按阶段隔离的速度图表 / **Real-time progress**: per-file speed, ETA, and phase-isolated speed charts
- **双语界面**：完整支持英文和中文 / **Bilingual UI**: full English and Chinese interface
- **预设系统**：保存并复用多目标工作流配置 / **Preset system**: save and recall multi-destination configurations

## 系统要求 | System Requirements

- macOS 12+（通用二进制：Apple Silicon + Intel x86_64） / macOS 12+ (Universal Binary: Apple Silicon + Intel x86_64)
- 下载体积约 13 MB / 13 MB download

## 技术指标 | Technical Stats

- Rust 后端：15 个模块，44 个 Tauri 命令 / Rust backend: 15 modules, 44 Tauri commands
- 119 个库测试 + 42 个集成测试（共 161 项），零失败 / 119 library tests + 42 integration tests (161 total), zero failures
- SQLite WAL 模式，16 张表结构 / SQLite WAL mode with a 16-table schema
- 生产路径零 `unsafe`，零 `unwrap` / Zero `unsafe` code and zero `unwrap` in production paths
