# v1.1.0-rc.2 — Windows 平台支持 / Windows Platform Support

## 新内容 | What's New

### Windows 平台支持 | Windows Platform Support
- 新增 Win32 API 卷检测（`GetDiskFreeSpaceExW`、`GetDriveTypeW`、`GetVolumeInformationW`、`GetLogicalDriveStringsW`） / Added Win32 API volume detection (`GetDiskFreeSpaceExW`, `GetDriveTypeW`, `GetVolumeInformationW`, `GetLogicalDriveStringsW`)
- 新增 PowerShell WMI 设备类型识别（SSD / HDD / USB / NVMe），并带有会话级缓存 / Added PowerShell WMI device-type detection (SSD / HDD / USB / NVMe) with per-session caching
- 新增 NSIS `.exe` 安装包构建 / Added NSIS `.exe` installer packaging
- 新增 Windows 风格应用菜单（File / Help） / Added a Windows-style application menu (File / Help)
- Debug bundle 适配所有平台（`cmd /c ver`、`wmic`、`GlobalMemoryStatusEx`） / Updated the debug bundle to work across platforms (`cmd /c ver`, `wmic`, `GlobalMemoryStatusEx`)
- GitHub Actions 新增 Windows CI（编译、测试、产物上传） / Added Windows CI in GitHub Actions for build, test, and artifact upload

### 其他改进 | Other Improvements
- 构建时间戳改为跨平台实现（用 `chrono` 替代 Unix `date` 命令） / Made build timestamps cross-platform by replacing the Unix `date` command with `chrono`
- 统一 MHL hostname 检测，移除 `#[cfg(unix)]` 限制 / Unified MHL hostname detection by removing the `#[cfg(unix)]` restriction
- Windows 环境增加 CJK 无衬线字体回退 / Added CJK sans-serif font fallback for Windows environments
- 官网新增稳定版 / 测试版双下载按钮 / Added separate stable and beta download buttons on the website
- 修复 Cascade phase 事件排序测试 / Fixed the Cascade phase event ordering test
