# v1.0.0-rc.2 — 第二个候选版 / Release Candidate 2

## 新功能 | New Features

- **SD / CF 卡设备类型识别**：新增紫色 SD 图标，可通过 `Protocol: Secure Digital` 或 USB 可移动介质自动识别 / **SD / CF card device type**: added a new purple SD icon with auto-detection via `Protocol: Secure Digital` or USB removable media
- **文件系统显示**：卷卡片现在会在设备类型旁显示格式（ExFAT、APFS、HFS+、SMB、NFS 等） / **File system display**: volume cards now show the file system format (ExFAT, APFS, HFS+, SMB, NFS, and more) next to the device type

## 问题修复 | Bug Fixes

- **SSD 被误判为 SD 卡**：优先检查 `Solid State: Yes`，确保像 A001 这样的 USB SSD 正确显示为 SSD / **SSD misclassified as SD**: now prioritizes `Solid State: Yes`, so USB SSDs like A001 are correctly shown as SSDs

## 统计 | Stats

- 117 个库测试通过（0 失败） / 117 library tests passing (0 failures)
- IO 调度器对 SD 卡调优为并发 2、缓冲 2 MB / IO scheduler tuned SD cards to 2-way concurrency with a 2 MB buffer
