# v1.0.0-beta.3 — 首次发布 / First Release

面向影视制作的专业卡数据卸载引擎 / Professional card offload engine for film production.

## 亮点 | Highlights

- **多目标复制**：支持并行写入与原子缓冲 / **Multi-destination copy** with parallel write and atomic buffering
- **哈希校验**：支持 XXH64 / XXH3 / SHA-256 / MD5，速度可达 10+ GB/s / **Hash verification** with XXH64 / XXH3 / SHA-256 / MD5 at 10+ GB/s
- **ASC MHL v2.0**：生成校验清单并保证链路完整性 / **ASC MHL v2.0** manifest generation and chain integrity
- **级联复制**：先写快速 SSD，再自动级联到慢速目标 / **Cascading copy** from fast SSD first to slower targets automatically
- **冲突检测**：支持 skip / overwrite / keep both 的逐文件策略 / **Conflict detection** with per-file skip / overwrite / keep both options
- **检查点与恢复**：可从崩溃、断线或断电中恢复 / **Checkpoint and resume** to survive crashes, cable pulls, and power loss
- **IO 调度器**：按设备类型设置并发（HDD 1-2 / SSD 4 / NVMe 8） / **IO scheduler** with per-device concurrency (HDD 1-2 / SSD 4 / NVMe 8)
- **实时进度**：速度图、ETA 和逐文件状态 / **Real-time progress** with speed chart, ETA, and per-file status
- **报告系统**：拍摄日汇总 + 任务明细（HTML / TXT） / **Reports** for shooting-day summary and job detail (HTML / TXT)
- **邮件通知**：完成或错误时支持 SMTP 通知 / **Email notifications** via SMTP on completion or error
- **工作流预设**：保存并复用卸载配置 / **Workflow presets** to save and reuse offload configs
- **国际化**：支持英文和中文 / **i18n** with English and Chinese
