# v1.1.0-rc.1 — 级联安全加固与体验打磨 / Cascade Safety Hardening and UX Polish

## 级联复制安全（关键） | Cascade Copy Safety (Critical)

- **P0 源卡可弹出条件控制**：只有主目标完整通过校验后才释放源卡；若重试后仍失败，会警告用户并保持源卡锁定 / **P0 SourceReleased conditional**: the source card is released only after the primary destination fully passes verification; if retry still fails, the user is warned and the source stays locked
- **P1 从主目标重试**：第 4b 阶段次级目标重试改为从已验证的主目标读取，而不是从可能已拔出的源卡读取 / **P1 Retry from primary**: Phase 4b secondary-destination retry now reads from the verified primary destination instead of the source card, which may already be ejected
- **级联覆盖策略**：消除静默的网络哈希行为，避免出现“复制中卡住”的挂起 / **Overwrite policy for cascade**: removes silent network hashing that caused "stuck during copying" hangs

## 文件处理 | File Handling

- **排除 AppleDouble（`._`）文件**：源扫描时过滤 macOS resource fork 文件，避免在 ExFAT / SD 卡上出现误报校验失败 / **AppleDouble (`._`) files excluded**: source scanning now filters macOS resource fork files to prevent false verification failures on ExFAT and SD media

## 体验改进 | UX Improvements

- 文件内校验进度每 300ms 更新，避免慢速介质上看起来“卡住” / Intra-file verification progress now updates every 300ms to avoid looking stuck on slow media
- 级联目标重排从拖拽改为箭头按钮 / Cascade destination reordering now uses arrow buttons instead of drag-and-drop
- “Speed priority” 重命名为 “Cascade Speed Priority”，并增加提示文案 / Renamed "Speed priority" to "Cascade Speed Priority" and added hint text
- 侧边栏导航新增官网按钮 / Added a website button to the sidebar navigation
- 阶段切换时重置进度，避免瞬间闪到 100% / Reset progress during phase transitions to avoid a flash to 100%
- ESLint 警告清零 / ESLint warnings reduced to zero
