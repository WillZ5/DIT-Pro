# v1.1.0-rc.2 — Windows Platform Support Pre-release

This release candidate introduces initial Windows platform support for DIT Pro.

## Highlights
- Added Windows desktop build and packaging support.
- Improved Windows compatibility across path handling and platform-specific behavior.
- Updated CI coverage for Windows release validation.

## Notes
- This is a pre-release (`rc.2`) intended for validation before stable release.
- Please report any Windows-specific issues discovered during testing.
