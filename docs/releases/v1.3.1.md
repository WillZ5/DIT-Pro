# v1.3.1 — 稳定版：素材日志、稳定性与性能优化 / Stable Release: Rushes Log, Stability & Performance

DIT Pro v1.3.1 是一个重大的稳定版更新，引入了影视后期流转的关键功能——素材日志（Rushes Log），并彻底修复了影响大容量存储安全的重要 Bug。

## 新功能 | New Features

### 1. Rushes Log (素材日志)
- **多格式导出**：支持 CSV、TSV、Excel (.xlsx) 和 PDF 格式导出拍摄日报告。
- **元数据提取**：自动探测视频文件的分辨率、编码、时码（Timecode）及色彩空间。
- **校验状态展示**：在日志中直观展示每个卷的哈希校验及 MHL 验证结果。

### 2. 核心稳定性与修复
- **机械硬盘容量修复 (HDD Capacity Fix)**：修复了 16TB 以上硬盘容量显示为 0 的溢出 Bug，确保大容量存储识别准确。
- **拷贝性能优化**：通过 IPC 事件限流和数据库锁竞争优化，解决了高吞吐量下可能出现的“掉速”问题，备份过程更平滑。
- **UI 交互增强**：优化了任务进度展示和长按退出（⌘Q）的视觉反馈。
