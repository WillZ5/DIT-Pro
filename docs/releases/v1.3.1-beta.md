# v1.3.1-beta — 稳定性与性能优化 / Stability & Performance Optimization

v1.3.1 是对 1.3 核心功能的关键维护版本，解决了机械硬盘容量显示异常及高吞吐量下的掉速问题。

## 修复与优化 | Fixes & Optimizations

- **机械硬盘容量修复 (HDD Capacity Fix)**：
  - 升级内部计算至 `u128`，解决了大容量（>16TB）机械硬盘在部分系统上显示为 0 字节的溢出 Bug。
  - 增强了 `statvfs` 数据校验的鲁棒性。

- **拷贝速度优化 (Copy Speed Optimization)**：
  - **事件限流**：将进度更新（IPC）频率从 100ms 调整为 500ms，大幅减轻了高并发下的 CPU 负载。
  - **锁竞争优化**：重构了媒体元数据（Media Metadata）探测逻辑，消除了任务开始时的“线程风暴”导致的数据库死锁风险。
  - **性能平滑**：优化了小文件备份时的状态更新流程，使拷贝过程更加稳健。

- **其他改进**：
  - 规范了版本分发渠道，更新了 `latest-beta.json` 自动更新配置。
