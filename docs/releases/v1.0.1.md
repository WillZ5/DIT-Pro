# v1.0.1 — 功能更新与修复 / Feature Updates and Fixes

## 新功能 | New Features

- **预检查剩余空间**：启动卸载前先扫描源大小并校验目标剩余空间 / **Pre-flight space check**: scans source size and verifies destination space before starting offload
- **源卡可弹出提示**：级联模式下，源卡可安全拔出时显示绿色提示 / **Source release notification**: shows a green notice when the source card is safe to eject in cascade mode
- **统一速度图**：所有阶段显示在同一时间轴上，并用颜色区分 / **Unified speed chart**: renders all phases on a single timeline with color-coded segments
- **更新检查器**：通过 GitHub API 自动检测版本，并以官网作为回退 / **Update checker**: detects new versions through the GitHub API with the website as a fallback

## 问题修复 | Bug Fixes

- 当启用 `post_verify` 时，`SourceReleased` 事件现在会在校验后正确延迟触发 / The `SourceReleased` event is now correctly delayed until after verification when `post_verify` is enabled
- 空间预检查失败会显示为可用空间为 0，而不是被静默跳过 / Space pre-check failures now surface as 0 available space instead of being silently skipped
- 速度图可以正确处理重复阶段（Copying → Verify → Copying retry），不会冲突 / The speed chart now handles repeated phases (Copying -> Verify -> Copying retry) without collisions
- 稳定版渠道下隐藏 Beta 警告对话框 / The beta warning dialog is now hidden for stable channel releases

## 技术细节 | Technical Details

- 新增 `reqwest`（`rustls-tls`）用于 HTTP 请求 / Added `reqwest` (`rustls-tls`) for HTTP requests
- 官网回退接口：`ditpro.negdims.com/software/latest.json` / Website fallback endpoint: `ditpro.negdims.com/software/latest.json`
- 119 个库测试通过，`clippy` 零警告 / 119 library tests passing with zero `clippy` warnings
